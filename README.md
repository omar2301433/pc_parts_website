## PC Parts E-Commerce Website

 A full-stack PC Parts E-Commerce Website built to help users browse, search, and purchase computer components such as CPUs, GPUs, RAM, storage devices, motherboards, and accessories.

## Features

- Browse PC components by category
- Search products by name
- Product details page
- Add to cart functionality
- User authentication (Login/Register)
- Admin dashboard for managing products
- Responsive design for desktop and mobile
- Order management system

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Template-based UI design

### Backend
- Node.js
- Express.js

### Database
- MongoDB


## Installation

1. Clone the repository

bash git clone https://github.com/omar2301433/pc_parts_website.git 

2. Navigate to project folder

bash cd pc-parts-website 

3. Install dependencies

bash npm install 

4. Run the server

bash npm start 

5. Open in browser

bash http://localhost:3000 

## Environment Variables

Create a .env file in the root directory and add:

env PORT=3000 MONGO_URI=your_mongodb_connection_string JWT_SECRET=your_secret_key
